Docs:       https://terminater.dev/docs
Donate:     https://terminater.dev/donate
Community:  https://terminater.dev/community

Working with packages:

 - Search:   pkg search <query>
 - Install:  pkg install <package>
 - Upgrade:  pkg upgrade

Subscribing to additional repositories:

 - Root:     pkg install root-repo
 - X11:      pkg install x11-repo

For fixing any repository issues,
try 'terminater-change-repo' command.

Report issues at https://terminater.dev/issues
